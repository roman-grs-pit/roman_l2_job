Phase 2 — skycell / SCA overlap

Finding depth-6 skycells for each selected pair and the exact (pointing, SCA) list to simulate

What Phase 2 does

Phase 2 is purely geometric — no image data is simulated. For each of the 30 pointings from Phase 1, we:

  1. Build the 18-SCA WCS at the pointing (via galsim.roman.getWCS(world_pos, SCAs=None, date, PA)), which applies the SIAF field-of-view offsets and the Roman distortion model to produce a GalSim CelestialWCS per SCA.
  2. Extract the four corner positions of each 4088×4088 SCA.
  3. Call romancal.skycell.match.find_skycell_matches(corners) to get the list of skycells the SCA footprint overlaps — this walks the skymap KDTree and tests polygon overlap against each candidate projection region’s skycell grid.

From the 540 (pointing, SCA) → skycell(s) triples we build two aggregate tables and pick one depth-6 skycell per pair for Phase 3/4.

Every pair has ≥ 1 skycell where all 6 pointings overlap (depth 6, the design maximum); depth falls off near the edges of the pair’s combined footprint.

## Overlap counts

Every pair has ≥ 1 skycell where all 6 pointings overlap
(**depth 6**, the design maximum); depth falls off near the edges of the
pair's combined footprint.

```{python}
#| echo: false
#| tbl-cap: "Per-pair skycell-depth distribution. A depth-d skycell gets contributions from d of the pair's 6 pointings."
depth_table = (PT_TABLE
               .groupby(["PAIR_ID", "n_pointings"]).size()
               .rename("n_skycells").reset_index()
               .pivot(index="PAIR_ID", columns="n_pointings",
                      values="n_skycells").fillna(0).astype(int))
depth_table.columns = [f"depth {c}" for c in depth_table.columns]
depth_table
```

## L2 files to simulate

For each selected skycell, the set of `(pointing, SCA)` pairs whose
footprint overlaps the skycell:

```{python}
#| echo: false
#| tbl-cap: "L2 file count per pair (distinct `(POINTING_IDX, SCA)` combinations after dedup)."
l2_counts = (NEED_SIM
             .drop_duplicates(["PAIR_ID", "POINTING_IDX", "SCA"])
             .groupby("PAIR_ID").size().reset_index(name="n_l2_files"))
l2_counts["n_pointings"] = NEED_SIM.groupby("PAIR_ID")["POINTING_IDX"].nunique().values
l2_counts["scas_per_pointing_mean"] = (l2_counts["n_l2_files"] /
                                       l2_counts["n_pointings"]).round(2)
l2_counts
```

## Self-checks

```{python}
#| echo: false
#| tbl-cap: "End-of-phase assertions on the overlap output."
checks = [
    ("5 skycells selected (one per pair)",        len(SELECTED) == 5),
    ("every selected skycell has depth 6",
     bool((SELECTED["n_pointings"] == 6).all())),
    ("every pair has exactly one selected skycell",
     SELECTED["PAIR_ID"].nunique() == SELECTED.shape[0]),
    ("every pointing appears in the SCA table",
     SCA_TABLE["POINTING_IDX"].nunique() == len(POINTINGS)),
    ("all selected skycells have ≥ 6 distinct L2 contributions",
     bool((SELECTED["n_sca_events"] >= 6).all())),
    ("n_sca_events ≥ n_pointings for every (pair, skycell)",
     bool((PT_TABLE["n_sca_events"] >= PT_TABLE["n_pointings"]).all())),
]
pd.DataFrame(checks, columns=["check", "pass"])
```
